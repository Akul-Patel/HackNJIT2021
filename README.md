```bash
  git clone -- Project
  install yarn
  install pip
  ------------------------
  yarn install [Installing Node Modules]
  pip install -r requirements.txt [Server side Library install in Venv]
  -------------------------------------------------------------------------
  yarn start [Start React Server]
  yarn start-api [Start Flask Server]
                or
  cd server
  flask start
```
    
## Tech Stack

**Client:** React, CSS

**Server:** Flask
